# Data visualization australian open
*Project Description*:
This project analyzes the Australian Open tennis dataset using advanced data visualization techniques to identify trends in player performance, nationalities, and match outcomes.

*Key Features*:
Treemap visualization for player distribution
Scatter plots for performance comparison
Geographic map of player nationalities
Time-based trend analysis
Dashboard created in Tableau

*Technologies Used*:
Tableau, Excel, Data Cleaning, Statistical Analysis

*Outcome*:
Provided insights into player performance patterns, dominance trends, and country-level participation in the Australian Open.
